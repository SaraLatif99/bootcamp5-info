- Product Overview

- Design Overview
    - UI/UX designers to share share wireframes & the current design screenshots
- Workshop 1:
    - Database fundamentals
    - Git, GitHub and the GitHub flow

## Week 2 Agenda

- Coding Session: Project Setup (Repo & CI/CD)
